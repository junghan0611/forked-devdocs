<dd>

<dd>는 정의 목록 요소(<dl>)에서 앞선 용어(<dt>)에 대한 설명, 정의, 또는 값을 제공합니다.

시도해보기

특성

이 요소는 전역 특성을 포함합니다.

nowrap 비표준

값이 yes면 너비의 끝에서 정의 텍스트의 줄이 바뀌지 않습니다. 기본값은 no입니다.
