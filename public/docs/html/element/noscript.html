<noscript>

<noscript>는 페이지의 스크립트 유형을 지원하지 않거나, 브라우저가 스크립트를 비활성화한 경우 보여줄 HTML 구획을 정의합니다.

특성

이 요소는 전역 특성만 포함합니다.

예제

html

<noscript>
  <!-- anchor linking to external file -->
  <a href="http://www.mozilla.com/">External Link</a>
</noscript>
<p>Rocks!</p>

스크립트 활성화 상태에서의 결과

Rocks!

스크립트 비활성화 상태에서의 결과

External Link

Rocks!
