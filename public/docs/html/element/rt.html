<rt>: 루비 텍스트 요소

<rt>는 동아시아 문자의 루비 주석에서 발음, 번역 등을 나타내는 텍스트 부분을 지정합니다.

시도해보기

특성

이 요소는 전역 특성만 포함합니다.

예제

다음 예제는 <ruby> 요소를 사용해 한자의 발음을 표기합니다.

html

<ruby>
  漢 <rt>한</rt>
  字 <rt>자</rt>
</ruby>
