<aside>: 별도 구획 요소

<aside>는 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다.

시도해보기

특성

이 요소는 전역 특성만 포함합니다.

사용 일람

  • 괄호에 묶인 텍스트라도 문서의 주요 플로우에 포함되어야 하므로 <aside>로 나타내선 안됩니다.

예제

<aside> 사용하기

다음 예제는 글 내의 문단을 <aside>로 표시합니다. 해당 문단은 글의 주제와 간접적으로만 연결되어 있습니다.

html

<article>
  <p>
    디즈니 만화영화 <em>인어 공주</em>는
    1989년 처음 개봉했습니다.
  </p>
  <aside>
    인어 공주는 첫 개봉 당시 8700만불의 흥행을 기록했습니다.
  </aside>
  <p>
    영화에 대한 정보...
  </p>
</article>
