Age

Age 헤더는 객체가 프록시 캐시 내에 머무는 초단위의 시간을 가집니다.

Age 헤더는 보통 0에 가깝습니다. 만약 Age: 0라면, 그것은 아마도 원 서버로부터 막 내려받은 것일 겁니다. 그게 아니라면 프록시의 현재 시간과 HTTP 응답 내에 포함된 Date 일반 헤더의 차로 계산됩니다.

구문

http

Age: <delta-seconds>

지시자

<delta-seconds>

음수가 아닌 정수형으로, 객체가 프록시 캐시 내에 머문 초단위 시간을 나타냅니다.

예제

Age: 24
