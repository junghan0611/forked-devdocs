GeolocationCoordinates: accuracy 속성

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

GeolocationCoordinates.accuracy 읽기 전용 속성은 GeolocationCoordinates.latitude와 GeolocationCoordinates.longitude의 오차범위(신뢰수준 95%)를 미터 단위로 표현한, 엄격히 양수로 제한된 double입니다.

값

GeolocationCoordinates.latitude와 GeolocationCoordinates.longitude의 오차범위(신뢰수준 95%)를 미터로 나타낸 양의 double입니다.
