GeolocationCoordinates: altitude 속성

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

GeolocationCoordinates.altitude 읽기 전용 속성은 평균 해수면을 정의하는 WGS84 타원체를 기준으로 했을 때, 이 위치의 고도를 미터 단위로 표현한 double입니다. 구현체가 고도 데이터를 제공하지 못하면 이 값은 null입니다.

값

WGS84 타원체를 기준으로 한 미터 단위의 고도를 나타내는 double입니다.
