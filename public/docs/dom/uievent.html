UIEvent

UIEvent 인터페이스는 간단한 사용자 인터페이스 이벤트를 나타냅니다.

UIEvent는 Event를 상속합니다. 하위 호환을 유지하기 위해 UIEvent.initMouseEvent() 메서드가 사라지지는 않았지만, UIEvent 객체의 생성은 UIEvent() 생성자로 해야 합니다.

MouseEvent, TouchEvent (en-US), FocusEvent (en-US), KeyboardEvent, WheelEvent (en-US), InputEvent, CompositionEvent (en-US) 인터페이스가 UIEvent를 직접 또는 간접적으로 상속합니다.

생성자

UIEvent()

UIEvent 객체를 생성합니다.

인스턴스 속성

Event 인터페이스의 속성을 상속합니다.

UIEvent.detail 읽기 전용

이벤트에 관련된 세부사항을 나타내는 long을 반환합니다. 정확한 정보는 이벤트 유형에 따라 다릅니다.

UIEvent.sourceCapabilities (en-US) Experimental 읽기 전용

InputDeviceCapabilities 인터페이스를 반환합니다. 터치 이벤트를 생성하는 물리적 장치에 대한 정보를 담고 있습니다.

UIEvent.view 읽기 전용

이벤트를 생성한 뷰를 담은 WindowProxy (en-US)를 반환합니다.

UIEvent.which (en-US) 지원이 중단되었습니다 읽기 전용

입력한 숫자 keyCode, 또는 알파벳 키의 경우 문자 코드(charCode)를 반환합니다.

인스턴스 메서드

Event 인터페이스의 메서드를 상속합니다.

UIEvent.initUIEvent() (en-US) 지원이 중단되었습니다

UIEvent 객체를 초기화합니다. 이미 발송한 이벤트에서는 아무 동작도 하지 않습니다.
