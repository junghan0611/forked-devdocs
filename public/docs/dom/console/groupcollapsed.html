console.groupCollapsed()

console.groupCollapsed() 메서드는 웹 콘솔에 새로운 인라인 그룹을 생성합니다. console.group()과는 달리, console.groupCollapsed()의 그룹은 닫힌 상태로 생성되며, 안쪽의 내용을 보려면 그룹 옆의 펼침 버튼으로 그룹을 먼저 펼쳐야 합니다.

console.groupEnd()를 호출해서 부모 그룹으로 나갈 수 있습니다.

자세한 내용과 예제는 console 문서의 콘솔 그룹 사용하기를 참고하세요.

참고: 이 기능은 Web Worker에서 사용할 수 있습니다

구문

js

groupCollapsed()
groupCollapsed(label)

매개변수

label Optional

그룹의 레이블입니다.
