ServiceWorkerRegistration.navigationPreload

Experimental: This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

ServiceWorkerRegistration 인터페이스의 navigationPreload 읽기 전용 속성은 현재의 서비스 워커 등록와 관련된 NavigationPreloadManager (en-US) 를 반환한다.

Syntax

js

var navigationPreloadManager = serviceWorkerRegistration.navigationPreload

Value

NavigationPreloadManager (en-US) 의 객체.
