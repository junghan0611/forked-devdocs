Document: copy 이벤트

copy 이벤트는 사용자가 브라우저의 사용자 인터페이스를 통해 복사했을 때 발생합니다.

copy 이벤트의 원 대상은 복사 동작의 대상인 Element입니다. Document에서 수신할 경우 캡처 또는 버블링 단계의 이벤트를 받게 됩니다. 이벤트에 대한 자세한 정보는 Element: copy 이벤트에서 확인하세요.

예제

js

document.addEventListener('copy', (event) => {
    console.log('복사 동작 실행')
});
