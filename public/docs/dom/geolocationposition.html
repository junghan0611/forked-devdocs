GeolocationPosition

Secure context: This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

GeolocationPosition 인터페이스는 주어진 시간에 장치가 위치한 지점을 나타냅니다. 지점은 Coordinates (en-US) 객체로 표현하여, 지구를 나타내는 회전타원체 위의 2D 위치와 더불어 높이와 속도 정보를 담습니다.

속성

GeolocationPosition 인터페이스는 어떤 속성도 상속하지 않습니다.

Position.coords (en-US) 읽기 전용 Secure context

주어진 시간의 위치를 나타내는 Coordinates (en-US) 객체입니다.

Position.timestamp (en-US) 읽기 전용 Secure context

위치를 기록한 시간을 나타내는 DOMTimeStamp (en-US)입니다.
