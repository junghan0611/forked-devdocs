UIEvent: view 속성

UIEvent.view 읽기 전용 속성은 이벤트가 생성된 WindowProxy (en-US) 객체를 반환합니다. 브라우저에서는 이벤트가 발생한 Window 객체입니다.

값

AbstractView 객체 참조입니다.
