MessageEvent

MessageEvent 는 WebSocket 또는 WebRTC RTCDataChannel (en-US) 으로 된 타겟으로 부터 전달받은 메시지를 보여주는 interface 입니다.

이 이벤트는 WebSocket.onmessage (en-US) 또는 RTCDataChannel.onmessage (en-US) 으로 설정된 이벤트 핸들러를 통해 실행이 되게 됩니다.

참고: 이 기능은 Web Worker에서 사용할 수 있습니다

생성자

MessageEvent() (en-US)

새로운 MessageEvent를 생성합니다.

속성

이 interface는 부모 객체인 Event의 속성을 상속받습니다.
