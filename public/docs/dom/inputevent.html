InputEvent

InputEvent 인터페이스는 편집 가능한 콘텐츠의 변경 이벤트를 나타냅니다.

생성자

InputEvent() (en-US)

InputEvent 객체를 생성합니다.

인스턴스 속성

UIEvent와 Event 인터페이스의 속성을 상속합니다.

InputEvent.data (en-US) 읽기 전용

삽입한 문자들로 구성된 문자열을 반환합니다. 문자를 삭제하는 등, 텍스트를 삽입하지 않는 변경 사항에서는 빈 문자열일 수 있습니다.

InputEvent.dataTransfer (en-US) 읽기 전용

편집 가능한 콘텐츠에 추가나 제거 중인 서식 텍스트 또는 평문 데이터의 정보가 포함된 DataTransfer 객체를 반환합니다.

InputEvent.inputType (en-US) 읽기 전용

편집 가능한 콘텐츠의 변경 유형을 반환합니다. 속성 문서에서 가능한 유형의 목록을 확인할 수 있습니다.

InputEvent.isComposing (en-US) 읽기 전용

이 이벤트가 compositionstart (en-US) 이후, compositionend (en-US) 이전에 발생했는지 나타내는 Boolean을 반환합니다.

인스턴스 메서드

UIEvent와 Event 인터페이스의 메서드를 상속합니다.

InputEvent.getTargetRanges() (en-US)

이 이벤트를 취소하지 않으면 영향을 받는 DOM 범위들의 배열을 반환합니다.
