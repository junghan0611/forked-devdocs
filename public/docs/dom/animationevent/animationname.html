AnimationEvent: animationName 속성

AnimationEvent.animationName 읽기 전용 속성은 트랜지션에 연관된 animation-name (en-US) CSS 속성 값을 담은 문자열입니다.

값

CSS animation-name (en-US) 속성의 값을 담은 문자열입니다.

  • 이 속성이 속한 AnimationEvent 인터페이스.
