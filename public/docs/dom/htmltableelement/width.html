table.width

요약

width는 바라는 테이블의 너비를 지정합니다.

구문

js

HTMLTableElement.width = width;
var width = HTMLTableElement.width;

width가 픽셀수나 퍼센트 값으로 너비를 나타내는 문자열인 곳.

예제

js

mytable.width = "75%";
