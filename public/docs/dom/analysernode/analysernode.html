AnalyserNode()

Web Audio API의 AnalyserNode() 생성자는 새로운 AnalyserNode 객체 인스턴스를 생성합니다.

구문

js

var analyserNode = new AnalyserNode(context, ?options);

매개변수

AudioNodeOptions (en-US) dictionary로부터 매개변수를 상속받습니다.

context

AudioContext 또는 OfflineAudioContext에의 참조.

options Optional

* fftSize: 주파수 영역 분석에 대한 FFT의 원하는 초기 사이즈. 기본값은 2048입니다.

  • maxDecibels: FFT 분석에 대한 dB단위로의 원하는 초기 최대 power. 기본값은 -30입니다.
  • minDecibels: FFT 분석에 대한 dB단위로의 원하는 초기 최소 power. 기본값은 -100입니다.
  • smoothingTimeConstant: FFT 분석에 대한 원하는 초기 smoothing 상수. 기본값은 0.8입니다.

반환 값

새로운 AnalyserNode 객체 인스턴스.
