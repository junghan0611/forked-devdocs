AudioWorkletProcessor.port

Experimental: This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

AudioWorkletProcessor 인터페이스의 읽기 전용 port 속성은 연관된 MessagePort (en-US)를 반환합니다. 이것은 프로세서와 프로세서가 속한 AudioWorkletNode 사이에서 소통하기 위해 사용될 수 있습니다.

참고: 채널의 다른 쪽 끝에 있는 포트는 노드의 port 속성 아래에서 사용 가능합니다.

구문

js

AudioWorkletProcessorInstance.port;

값

AudioWorkletProcessor 와 연관된 AudioWorkletNode 를 연결하고 있는 MessagePort (en-US) 객체.

예제

AudioWorkletNode.port에서 예제 코드를 볼 수 있습니다.
