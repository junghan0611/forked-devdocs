DOMRect

DOMRect 인터페이스는 직사각형의 크기와 위치를 나타냅니다.

DOMRect가 나타내는 사각형의 유형은 DOMRect를 반환한 메서드나 속성이 지정합니다. 예를 들어 WebVR API의 VREyeParameters.renderRect (en-US)는, 헤드 마운트 디스플레이에서 한쪽 눈의 뷰포트가 캔버스의 어디에 위치하는지 나타냅니다.

부모인 DOMRectReadOnly (en-US)을 상속합니다.

(en-US) DOMRect

생성자

DOMRect()

새로운 DOMRect 객체를 생성합니다.

속성

부모인 DOMRectReadOnly (en-US)의 속성을 상속합니다. 읽기 전용 속성이 아니라는 차이점이 있습니다.

DOMRectReadOnly.x (en-US)

DOMRect 원점의 X 좌표입니다. 원점은 보통 직사각형의 왼쪽 상단 꼭지점입니다.

DOMRectReadOnly.y (en-US)

DOMRect 원점의 Y 좌표입니다. 원점은 보통 직사각형의 왼쪽 상단 꼭지점입니다.

DOMRectReadOnly.width (en-US)

DOMRect의 너비입니다.

DOMRectReadOnly.height (en-US)

DOMRect의 높이입니다.

DOMRectReadOnly.top (en-US)

DOMRect 위쪽 모서리의 Y 좌표입니다. y 또는, height가 음수인 경우, y + height와 같은 값입니다.

DOMRectReadOnly.right (en-US)

DOMRect 오른쪽 모서리의 X 좌표입니다. x 또는, width가 음수인 경우, x + height와 같은 값입니다.

DOMRectReadOnly.bottom (en-US)

DOMRect 아래쪽 모서리의 Y 좌표입니다. y + height 또는, height가 음수인 경우, y와 같은 값입니다.

DOMRectReadOnly.left (en-US)

DOMRect 왼쪽 모서리의 X 좌표입니다. x + width 또는, width가 음수인 경우, x와 같은 값입니다.

메서드

부모인 DOMRectReadOnly (en-US)의 메서드를 상속합니다.

정적 메서드

DOMRectReadOnly.fromRect() (en-US)

주어진 위치와 면적을 갖는 새로운 DOMRect 객체를 생성합니다.
