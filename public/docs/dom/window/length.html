Window.length

window의 frame 개수를 반환합니다. (<frame> 또는 <iframe> 요소들 중 하나)

신텍스

js

framesCount = window.length;
  • framesCount 는 frame의 개수이다.

예제

js

if (window.length) {
  // document의 subframes
}
