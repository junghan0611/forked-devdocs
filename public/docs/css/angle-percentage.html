<angle-percentage>

CSS <angle-percentage> 자료형은 <angle>과 <percentage> 모두 가능한 값을 나타냅니다.

구문

<angle>과 <percentage> 문서에서 각 자료형의 구문을 살펴보세요.

calc()에서 사용하기

<angle-percentage> 를 사용할 수 있는 곳이라면 백분율이 각도로 계산되므로, 이 자료형도 calc()표현식에서 사용할 수 있습니다.
