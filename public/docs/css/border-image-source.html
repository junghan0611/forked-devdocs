border-image-source

border-image-source CSS 속성은 요소의 테두리 이미지로 사용할 원본 이미지를 지정합니다.

시도해보기

border-image-slice 속성으로 나뉜 구역을 사용해 최종 테두리 이미지를 생성합니다.

구문

css

/* 키워드 값 */
border-image-source: none;

/* <image> 값 */
border-image-source: url(image.jpg);
border-image-source: linear-gradient(to top, red, yellow);

/* 전역 값 */
border-image-source: inherit;
border-image-source: initial;
border-image-source: unset;

값

none

테두리 이미지를 사용하지 않습니다. border-style이 대신 표시됩니다.

<image>

테두리에 사용할 이미지 참조입니다.

형식 구문

border-image-source = 
  none     |
  <image>  

<image> = 
  <url>       |
  <gradient>  

<url> = 
  url( <string> (en-US) <url-modifier>* )  |
  src( <string> (en-US) <url-modifier>* )

예제

border-image에서 확인할 수 있습니다.
