Math.SQRT1_2

Math.SQRT1_2 프로퍼티는 약 0.707의 값을 나타내는 루트 1/2을 나타냅니다:

Math.SQRT1_2 = 1 2 = 1 2 ≈ 0.707 \mathtt{\mi{Math.SQRT1_2}} = \sqrt{\frac{1}{2}} = \frac{1}{\sqrt{2}} \approx 0.707

시도해보기

설명

SQRT1_2 는 Math의 정적 프로퍼티이기 때문에, 당신이 생성한 Math 오브젝트의 프로퍼티보다는 항상 Math.SQRT1_2의 형태로 사용합니다 (Math는 컨스트럭터가 아닙니다).

예시

Math.SQRT1_2 의 사용

다음은 루트 1/2를 출력합니다:

js

function getRoot1_2() {
  return Math.SQRT1_2;
}

getRoot1_2(); // 0.7071067811865476
