Math.exp()

**Math.exp()**함수는 x를 인수로 하는 e^x 값을 반환합니다. 그리고 e는 오일러 상수(또는 네이피어 상수)는 자연 로그의 밑입니다.

시도해보기

구문

js

    Math.exp(x)

매개변수

x

숫자

반환 값

e는 오일러 상수이고 x는 인수인 e^x값

Description

exp()는 Math의 정적 메소드이기 때문에 새로 작성한 Math 오브젝트 대신에 항상 Math.exp()의 형태로 써야 합니다. (Math는 생성자가 아닙니다.)

예시

Math.exp() 사용 예

js

Math.exp(-1); // 0.36787944117144233
Math.exp(0);  // 1
Math.exp(1);  // 2.718281828459045
