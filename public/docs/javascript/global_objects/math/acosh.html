Math.acosh()

Math.acosh() 정적 메서드는 숫자의 역 쌍곡선 코사인를 반환합니다. 아래와 같습니다.

∀ x ≥ 1 , 𝙼𝚊𝚝𝚑.𝚊𝚌𝚘𝚜𝚑 ( 𝚡 ) = arcosh ( x ) = the unique y ≥ 0 such that cosh ( y ) = x = ln ( x + x 2 − 1 ) \begin{aligned}\forall x \geq 1,;\mathtt{\operatorname{Math.acosh}(x)} &= \operatorname{arcosh}(x) = \text{the unique } y \geq 0 \text{ such that } \cosh(y) = x\&= \ln\left(x + \sqrt{x^2 - 1}\right)\end{aligned}

시도해보기

구문

js

Math.acosh(x)

매개변수

x

1보다 크거나 같은 숫자.

반환 값

x의 역 쌍곡선 코사인. x가 1보다 작으면 NaN를 반환합니다.

설명

acosh()는 Math의 정적 메서드이기 때문에, 생성한 Math 객체 메서드보다 Math.acosh()를 항상 사용합니다 (Math는 생성자가 아닙니다).

예제

Math.acosh() 사용

js

Math.acosh(0); // NaN
Math.acosh(1); // 0
Math.acosh(2); // 1.3169578969248166
Math.acosh(Infinity); // Infinity
