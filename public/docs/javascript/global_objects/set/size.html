Set.prototype.size

size 접근자 속성은 Set 객체의 원소 수를 반환합니다.

시도해보기

설명

size의 값은 Set 객체가 가진 원소의 수를 나타내는 정수입니다. size 값의 설정자는 undefined입니다. 즉 값을 직접 바꿀 수는 없습니다.

예제

size 사용하기

js

var mySet = new Set();
mySet.add(1);
mySet.add(5);
mySet.add('some text')

mySet.size; // 3
