Set.prototype.keys()

keys() 메서드는 values() 메서드의 별칭입니다.

구문

js

keys()

반환 값

주어진 Set의 각 요소에 대한 값을 포함하는 새로운 반복 객체입니다. 반복은 삽입 순입니다.
