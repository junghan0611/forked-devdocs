Map.prototype.keys()

keys() 메서드는 배열에서 각 인덱스의 키를 포함하는 새로운 맵 반복자 (en-US) 객체를 반환합니다.

시도해보기

구문

js

keys()

반환 값

새로운 순회 가능한 반복자 객체 (en-US).

예제

keys() 사용하기

js

const myMap = new Map();
myMap.set("0", "foo");
myMap.set(1, "bar");
myMap.set({}, "baz");

const mapIter = myMap.keys();

console.log(mapIter.next().value); // "0"
console.log(mapIter.next().value); // 1
console.log(mapIter.next().value); // {}
