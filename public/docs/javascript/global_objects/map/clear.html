Map.prototype.clear()

clear() 메서드는 Map 객체의 모든 요소를 제거합니다.

시도해보기

구문

js

clear()

반환 값

undefined.

예제

clear() 사용하기

js

const myMap = new Map();
myMap.set('bar', 'baz');
myMap.set(1, 'foo');

console.log(myMap.size);  // 2
console.log(myMap.has('bar')); // true

myMap.clear();

console.log(myMap.size);  // 0
console.log(myMap.has('bar')); // false
