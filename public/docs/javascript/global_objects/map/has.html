Map.prototype.has()

has() 메서드는 주어진 키에 해당하는 요소가 존재 여부를 가리키는 불리언 값을 반환합니다.

시도해보기

구문

js

has(key)

매개변수

key

Map 객체에서 존재를 확인할 요소의 키

반환 값

만약 주어진 키에 해당하는 요소가 Map객체에 존재한다면 true, 그렇지 않으면 false

예제

has() 사용하기

js

const myMap = new Map();
myMap.set("bar", "foo");

console.log(myMap.has("bar")); // true
console.log(myMap.has("baz")); // false
