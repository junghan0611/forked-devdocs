Number.NaN

Number.NaN 속성은 Not-A-Number(숫자가 아님)를 나타냅니다. NaN과 같습니다.

시도해보기

정적 속성이므로 접근하기 위해 Number 객체를 생성할 필요는 없습니다. (Number.NaN 사용)

예제

값이 숫자형인지 확인하기

js

function sanitise(x) {
  if (isNaN(x)) {
    return Number.NaN;
  }
  return x;
}
