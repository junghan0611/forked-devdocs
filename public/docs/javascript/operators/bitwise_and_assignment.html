비트 AND 할당 연산 (&=)

&= 연산자는 두 피연산자의 이진 표현을 사용하고, 피연산자에 대한 AND 비트 연산 수행 결과를 변수에 할당합니다.

시도해보기

구문

js

x &= y // x = x & y

예제

비트 AND 할당 연산 사용하기

js

let a = 5;
// 5:     00000000000000000000000000000101
// 2:     00000000000000000000000000000010
a &= 2; // 0
