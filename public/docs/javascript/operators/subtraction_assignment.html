빼기 할당 (-=)

빼기 할당 연산자(-=)는 오른쪽 피연산자의 값을 변수에서 뺀 결과를 다시 변수에 할당합니다.

시도해보기

구문

js

x -= y // x = x - y

예제

빼기 할당 사용하기

js

// bar = 5
// 위와 같은 변수를 가정할 때

bar -= 2     // 3
bar -= 'foo' // NaN
