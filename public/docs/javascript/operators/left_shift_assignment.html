왼쪽 시프트 할당 연산자 (<<=)

<<= 연산자는 지정된 비트 수 만큼 왼쪽으로 비트를 이동시키고 그 결과를 변수에 반영합니다.

시도해보기

구문

js

x <<= y // x = x << y

예제

왼쪽 시프트 할당하기

js

let a = 5;
// 00000000000000000000000000000101
a <<= 2; // 20
// 00000000000000000000000000010100
